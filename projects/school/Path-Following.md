# Path-Following

**Repository:** [github.com/CSW04-exe/Path-Following](https://github.com/CSW04-exe/Path-Following)  
**Language:** Python

## Description

Simulates 2D path-following steering behavior, where an autonomous agent tracks a predefined path by projecting ahead onto the nearest point of the path and computing a seek target ahead of that point.

## Technologies

- Python
- Pygame
- Vector mathematics

## Key Concepts

- Path-following steering behavior
- Future position prediction
- Nearest-point projection onto a path segment
- Combining with Arrive/Seek for smooth tracking
- Autonomous agent simulation
