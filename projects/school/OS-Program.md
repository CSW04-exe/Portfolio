# OS-Program

**Repository:** [github.com/CSW04-exe/OS-Program](https://github.com/CSW04-exe/OS-Program)  
**Language:** Java

## Description

Simulates a priority-based process scheduler, modeling core operating system scheduling concepts. Processes are queued, prioritized, and executed according to scheduling policies.

## Technologies

- Java
- Priority queues / data structures

## Key Concepts

- Process scheduling algorithms (Priority, FCFS, Round Robin concepts)
- Priority queue implementation
- Context switching simulation
- CPU burst and wait time tracking
- Turnaround time and average wait time metrics
