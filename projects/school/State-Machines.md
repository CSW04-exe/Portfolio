# State-Machines

**Repository:** [github.com/CSW04-exe/State-Machines](https://github.com/CSW04-exe/State-Machines)  
**Language:** Python

## Description

Implements a finite state machine simulation for vehicle passing scenarios. Vehicles transition between states (e.g., following, overtaking, merging back) based on conditions and triggers, modeling real traffic decision logic.

## Technologies

- Python
- State machine design pattern

## Key Concepts

- Finite state machines (FSM)
- State transitions and trigger conditions
- Event-driven behavior modeling
- Traffic / vehicle scenario simulation
- Clean separation of state logic
