# Path-Finding

**Repository:** [github.com/CSW04-exe/Path-Finding](https://github.com/CSW04-exe/Path-Finding)  
**Language:** Python

## Description

Implements the A* pathfinding algorithm on a game graph. Uses a heuristic function to guide the search toward the goal efficiently, finding the optimal path while minimizing explored nodes compared to uninformed search.

## Technologies

- Python
- heapq (priority queue)
- Pygame (optional visualization)

## Key Concepts

- A* search algorithm: f(n) = g(n) + h(n)
- Heuristic design (Manhattan / Euclidean distance)
- Graph representation (adjacency list / grid)
- Open set / closed set management
- Comparison with Dijkstra and BFS
