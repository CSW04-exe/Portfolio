A matrix calculator implemented in Java using object-oriented design principles. Supports standard matrix operations via a clean class hierarchy.

## Technologies

- Java
- Object-Oriented Programming

## Key Concepts

- Classes, inheritance, and encapsulation in Java
- Matrix operations: addition, subtraction, multiplication, transpose
- Input validation and error handling
- Object-oriented design patterns
