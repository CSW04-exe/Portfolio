# Multivariate-Linear-Regression

**Repository:** [github.com/CSW04-exe/Multivariate-Linear-Regression](https://github.com/CSW04-exe/Multivariate-Linear-Regression)  
**Language:** Python

## Description

Implements multivariable linear regression on the Boston housing dataset, predicting house prices from multiple features. Covers both a from-scratch implementation and a scikit-learn comparison.

## Technologies

- Python
- NumPy
- pandas
- scikit-learn
- matplotlib

## Key Concepts

- Multivariable linear regression
- Normal equation and gradient descent
- Feature scaling / normalization
- R², MSE, RMSE evaluation metrics
- Multicollinearity and feature correlation
