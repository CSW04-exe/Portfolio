# Univariate-Linear-Regression

**Repository:** [github.com/CSW04-exe/Univariate-Linear-Regression](https://github.com/CSW04-exe/Univariate-Linear-Regression)  
**Language:** Python

## Description

Demonstrates single-variable linear regression with worked examples. Covers the math behind fitting a line to data, implementing gradient descent, and evaluating the model — a foundational exercise in supervised machine learning.

## Technologies

- Python
- NumPy
- matplotlib

## Key Concepts

- Simple linear regression: y = mx + b
- Cost function (MSE)
- Gradient descent optimization
- Learning rate effects
- Model evaluation: R², residuals
