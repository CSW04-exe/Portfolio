# AI-MLProj

**Repository:** [github.com/CSW04-exe/AI-MLProj](https://github.com/CSW04-exe/AI-MLProj)  
**Language:** Python

## Description

Builds and compares two machine learning models — an Artificial Neural Network (ANN) and a Support Vector Machine (SVM) — on the same binary classification problem. The project evaluates and contrasts the performance metrics of both approaches to illustrate their strengths and trade-offs.

## Technologies

- Python
- TensorFlow / Keras (ANN)
- scikit-learn (SVM, metrics)
- NumPy / pandas

## Key Concepts

- Binary classification
- Neural network architecture and training
- Kernel-based classification (SVM)
- Accuracy, precision, recall, F1-score comparison
- Overfitting vs. generalization
