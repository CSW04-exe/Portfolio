# Logistic-Regression

**Repository:** [github.com/CSW04-exe/Logistic-Regression](https://github.com/CSW04-exe/Logistic-Regression)  
**Language:** Python

## Description

Implements binary logistic regression from scratch and via scikit-learn on the classic Iris dataset. Focuses on classification boundaries, sigmoid activation, and model evaluation.

## Technologies

- Python
- NumPy
- scikit-learn
- matplotlib

## Key Concepts

- Logistic regression (binary classification)
- Sigmoid / logistic function
- Log-likelihood and cross-entropy loss
- Decision boundary visualization
- Confusion matrix, accuracy, precision, recall
