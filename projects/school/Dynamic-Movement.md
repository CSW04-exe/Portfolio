# Dynamic-Movement

**Repository:** [github.com/CSW04-exe/Dynamic-Movement](https://github.com/CSW04-exe/Dynamic-Movement)  
**Language:** Python

## Description

Simulates 2D autonomous agent motion using classical AI steering behaviors. Implements Seek, Flee, and Arrive behaviors that produce realistic, smooth movement by computing steering forces based on the difference between desired and current velocity vectors.

## Technologies

- Python
- Pygame
- Vector mathematics

## Key Concepts

- Steering behaviors (Seek, Flee, Arrive)
- Autonomous agent simulation
- 2D vector math: velocity, acceleration, steering force
- Arrival deceleration / slowing radius
- Real-time simulation loop
