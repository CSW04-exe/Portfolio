# GO_Lang-Program

**Repository:** [github.com/CSW04-exe/GO_Lang-Program](https://github.com/CSW04-exe/GO_Lang-Program)  
**Language:** Go

## Description

Reads basketball player statistics from a data source and computes advanced shooting efficiency metrics — Effective Field Goal Percentage (eFG%) and True Shooting Percentage (TS%) — demonstrating Go's suitability for data processing tasks.

## Technologies

- Go (Golang)
- Standard library I/O and string handling

## Key Concepts

- File I/O and CSV/text parsing in Go
- Mathematical formulas: eFG% and TS%
- Structs and methods in Go
- Idiomatic Go formatting and error handling

## Formulas

- **eFG%** = (FG + 0.5 × 3P) / FGA  
- **TS%** = PTS / (2 × (FGA + 0.44 × FTA))
