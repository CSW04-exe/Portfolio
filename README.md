# Portfolio site

This branch adds a static portfolio site scaffold that lists your GitHub projects with README excerpts available on demand.

Files added:
- index.html
- styles.css
- script.js
- data/projects.json
- assets/avatar-placeholder.svg

How to preview locally
1. Clone the repository and checkout the branch `add/portfolio-site`.
2. Open `index.html` in a browser (no server required) or run a static server: `python -m http.server`.

How README fetching works
- The site attempts to load each repo's README from raw.githubusercontent.com using `main` then `master` branch. If a README is not accessible publicly it will display a fallback message.

To publish
- After merging, enable GitHub Pages in repository settings (choose `main` branch or `gh-pages` depending on workflow) or move files into `docs/` on the default branch.
